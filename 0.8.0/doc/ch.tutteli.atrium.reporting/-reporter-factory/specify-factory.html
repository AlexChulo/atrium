specifyFactory

fun specifyFactory(reporterFactoryId: String): Unit (source)

Sets Atrium's property with key ATRIUM_PROPERTY_KEY (which is ch.tutteli.atrium.reporting.reporterFactory) to the given reporterFactoryId regardless if another id was specified before.

Use specifyFactoryIfNotYetSet if you only want to set a default value but not overwrite an existing specification.
