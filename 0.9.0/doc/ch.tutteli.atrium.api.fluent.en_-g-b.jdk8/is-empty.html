isEmpty

fun <T : Optional<*>> Expect<T>.isEmpty(): Expect<T>

Expects that the subject of the assertion (an Optional) is empty (not present).

Shortcut for more or less something like feature(Optional<T>::isEmpty) { toBe(true) } depends on the underlying implementation though.

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
This assertion container to support a fluent API.
