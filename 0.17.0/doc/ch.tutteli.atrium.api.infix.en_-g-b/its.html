its

inline val <T> Expect<T>.its: Expect<T> (source)

Inline property referring actually to this and allows to write infix assertions within an assertion group block

For instance, instead of:

expect(person) {
  this name toBe 1
}

You can write

expect("hello world") {
  its name toBe 1
}

Return
this
