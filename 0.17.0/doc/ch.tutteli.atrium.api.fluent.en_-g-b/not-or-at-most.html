fun <T : CharSequence, S : CharSequenceContains.SearchBehaviour> CharSequenceContains.EntryPointStep<T, S>.notOrAtMost(times: Int): NotOrAtMostCheckerStep<T, S> (source)

Restricts a contains assertion by specifying that the number of occurrences of the value which we are looking for occurs not at all or at most number of times within the search input.

Parameters

times - The number which the check will compare against the actual number of times an expected value is found in the input of the search.

Exceptions

IllegalArgumentException - In case times is smaller than zero.

IllegalArgumentException - In case times is zero; use notToContain instead.

Return
The newly created builder.
