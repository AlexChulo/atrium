isNoneOf

fun <T> Expect<T>.isNoneOf(expected: T, vararg otherValues: T): Expect<T> (source)
Deprecated: Use notToEqualOneOf; will be removed with 1.0.0 at the latest

Expects that the subject of this expectation is not (equal to) expected and otherValues.

expect(99).isNoneOf(1, 2, 3, 4)

fails {
    expect(1).isNoneOf(1, 2, 3, 4)
}

Return
an Expect for the subject of this expectation.
