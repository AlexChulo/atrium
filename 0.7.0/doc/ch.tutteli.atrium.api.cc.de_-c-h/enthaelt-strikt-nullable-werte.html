enthaeltStriktNullableWerte

fun <E, T : Iterable<E>> Assert<T>.enthaeltStriktNullableWerte(expectedOrNull: E, vararg otherExpectedOrNulls: E): AssertionPlant<T> (source)

Makes the assertion that AssertionPlant.subject (which has a nullable entry type) contains only the expected nullable value and the other expected nullable values (if given) in the defined order.

It is a shortcut for enthaelt.inGegebenerReihenfolge.nur.nullableWerte(expectedOrNull, *otherExpectedOrNulls)

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
This plant to support a fluent API.
