newMethodCallFormatter

abstract fun newMethodCallFormatter(): MethodCallFormatter (source)

Creates a MethodCallFormatter which represents arguments of a method call by using their Object.toString representation with the exception of:

  • CharSequence, is wrapped in quotes (") and line breaks (CR or/and LF) are escaped so that the whole representation remains on one line.
  • Char is wrapped in apostrophes (')

Return
The newly created method call formatter.
