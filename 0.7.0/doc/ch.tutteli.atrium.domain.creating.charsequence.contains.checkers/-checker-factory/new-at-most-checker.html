newAtMostChecker

abstract fun newAtMostChecker(times: Int, nameContainsNotFun: String, atMostCall: (Int) -> String): Checker (source)

Creates an CharSequenceContains.Checker which verifies that an expected object is contained at most times in the search input.

Parameters

times - The number which the check uses to compare against the actual number of times an expected object is found in the input of the search.

nameContainsNotFun - The function which should be used instead of atMostCall when times equals to zero.

atMostCall - The function which was used and should not be used if times equals to zero.

Exceptions

IllegalArgumentException - In case times is smaller than 1.
