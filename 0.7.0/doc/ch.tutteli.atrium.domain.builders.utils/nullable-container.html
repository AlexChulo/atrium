nullableContainer

inline fun <T> nullableContainer(iterable: Iterable<T>): Iterable<T?> (source)

Turns an Iterable into an iterable with a nullable entry type.

Intended to be used in conjunction with platform types such as Iterable<String!> or in other words, when you deal with Java and you want to turn a platform type into a nullable type.

Basically it is a replacement for cast but without introducing one explicitly (we only give an additional hint to the compiler that we expect a nullable type and not a non-nullable one). For instance, instead of writing getPersons() as List<String?> you can write nullableContainer(getPersons())
