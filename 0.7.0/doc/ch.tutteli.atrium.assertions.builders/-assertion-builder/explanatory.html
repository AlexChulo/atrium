explanatory

abstract val explanatory: ExplanatoryAssertionExplanationOption (source)

Builder to create an ExplanatoryAssertion -- use it to explain something which is typically formatted by an ObjectFormatter.

For instance, it is used to explain additional entries in an Iterable contains entries assertion. It is typically used in an explanatoryGroup.
