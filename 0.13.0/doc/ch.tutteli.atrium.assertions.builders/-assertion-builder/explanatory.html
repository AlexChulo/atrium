explanatory

abstract val explanatory: Explanatory.ExplanationOption (source)

Builder to create an ExplanatoryAssertion -- use it to explain something which is typically formatted by an ObjectFormatter -- has to be a child of an ExplanatoryGroup (see explanatoryGroup).

For instance, it is used to explain additional entries in an Iterable contains entries assertion.

Use representationOnly instead if the assertion can also fail.
