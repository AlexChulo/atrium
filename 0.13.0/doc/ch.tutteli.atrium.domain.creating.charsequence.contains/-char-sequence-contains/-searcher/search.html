search

abstract fun search(searchIn: CharSequence, searchFor: SC): Int (source)

Searches in the given searchIn for the given searchFor, using its toString implementation, and returns the number of occurrences.

Whether searches are disjoint or non-disjoint is up to the implementation.

Parameters

searchIn - The input CharSequence in which this Searcher shall search

searchFor - The object which shall be found

Return
The number of occurrences of searchFor in searchIn.
