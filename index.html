Overview

  • Atrium is an open-source assertion library for Kotlin.
  • It is licences under EUPL 1.2
  • You can post questions in the Discussion tab of the project.
  • Jump straight to the latest version (which is 0.18.0)
  • ... or use the version switcher above to see the documentation of another version.

You are very welcome to contribute to Atrium:

  • star Atrium if you like it and spread the word.
  • open a bug or create a feature request
  • Take a look at CONTRIBUTING.md for more ideas.
