isEqualComparingTo

infix fun <T : Comparable<T>> Expect<T>.isEqualComparingTo(expected: T): Expect<T> (source)

Expects that the subject of this expectation is equal to expected where the comparison is carried out with Comparable.compareTo.

expect(2) isEqualComparingTo 2

fails {
    expect(1) isEqualComparingTo 2
    expect(2) isEqualComparingTo 1
}

Return
an Expect for the subject of this expectation.
