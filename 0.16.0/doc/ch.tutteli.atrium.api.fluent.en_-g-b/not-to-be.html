notToBe

fun <T> Expect<T>.notToBe(expected: T): Expect<T> (source)

Expects that the subject of this expectation is not (equal to) expected.

expect(2).notToBe(3)

fails {
    expect(12).notToBe(12)
}

expect(listOf(2)).notToBe(listOf(3))

fails {
    expect(listOf(2)).notToBe(listOf(2))
}

Return
an Expect for the subject of this expectation.
