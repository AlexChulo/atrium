DefaultIterableLikeToIterableTransformer()

Transforms Sequence and all Array types to Iterable.

following a more precise list:

  • Iterable (does not transform anything)
  • Sequence
  • Array
  • CharArray
  • ByteArray
  • ShortArray
  • IntArray
  • LongArray
  • FloatArray
  • DoubleArray
  • BooleanArray
