buildChained

inline fun <reified I : Any> ComponentFactoryContainer.buildChained(): Sequence<I> (source)

Returns a chain of components of type I using a corresponding factory or throws an IllegalStateException in case no factory was found which is able to build a chain of components of the given type.

Exceptions

IllegalStateException - in case ComponentFactoryContainer.buildChainedOrNull returns null because no suitable factory was found.

ClassCastException - in case one of factories returns an illegal type.
