isLessThanOrEqual

infix fun <T : Comparable<T>> Expect<T>.isLessThanOrEqual(expected: T): Expect<T> (source)
Deprecated: Use toBeLessThanOrEqualTo; will be removed with 1.0.0 at the latest

Expects that the subject of this expectation is less than or equal expected. The comparison is carried out with Comparable.compareTo.

expect(1) isLessThanOrEqual 2
expect(2) isLessThanOrEqual 2

fails {
    expect(2) isLessThanOrEqual 1
}

Return
an Expect for the subject of this expectation.
