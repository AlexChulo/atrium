isGreaterThan

infix fun <T : Comparable<T>> Expect<T>.isGreaterThan(expected: T): Expect<T> (source)
Deprecated: Use toBeGreaterThan; will be removed with 1.0.0 at the latest

Expects that the subject of this expectation is greater than expected. The comparison is carried out with Comparable.compareTo.

expect(2) isGreaterThan 1

fails {
    expect(2) isGreaterThan 2
}

Return
an Expect for the subject of this expectation.
