isGreaterThanOrEqual

infix fun <T : Comparable<T>> Expect<T>.isGreaterThanOrEqual(expected: T): Expect<T> (source)
Deprecated: Use toBeGreaterThanOrEqualTo; will be removed with 1.0.0 at the latest

Expects that the subject of this expectation is greater than or equal expected. The comparison is carried out with Comparable.compareTo.

expect(2) isEqualComparingTo 2

fails {
    expect(1) isEqualComparingTo 2
    expect(2) isEqualComparingTo 1
}

Return
an Expect for the subject of this expectation.
