notToEqualOneOf

fun <T> Expect<T>.notToEqualOneOf(expected: T, vararg otherValues: T): Expect<T> (source)

Expects that the subject of this expectation is not (equal to) expected and otherValues.

expect(99).notToEqualOneOf(1, 2, 3, 4)

fails {
    expect(1).notToEqualOneOf(1, 2, 3, 4)
}

Return
an Expect for the subject of this expectation.
