fun <T : CharSequence> Expect<T>.notToBeEmpty(): Expect<T> (source)

Expects that the subject of this expectation (a CharSequence) CharSequence.kotlin.text.isNotEmpty.

fails {
    expect("").notToBeEmpty()
}

// use notToBeBlank to check for whitespaces
